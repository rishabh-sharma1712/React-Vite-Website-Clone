# React + Vite + Tailwind Website Clone

### Prerequisites
- Node.js
- npm

### Installation

1. Clone the repository:
2. Navigate into the project directory:
  - cd project-directory
3. Install the dependencies:
 - npm install
4. Start the development server:
 - npm run dev

## Technologies Used
- React.js
- Vite
- Tailwind
